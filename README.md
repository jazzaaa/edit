# LGP Social Generator

Use this GitHub repo to generate either a Reel video or a poster.

## Each time

1. Put 4 photos in `photos/`
2. Name them:
   - photo1.jpg
   - photo2.jpg
   - photo3.jpg
   - photo4.jpg

It also supports `.png`, `.webp`, `.avif`, `.jpeg`.

3. Edit `texts.json`
4. Go to GitHub → Actions

For video:
- Run `Render LGP Reel`

For poster:
- Run `Render LGP Poster`
